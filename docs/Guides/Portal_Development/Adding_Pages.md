# Adding Pages

To add pages, just add `.html` or `.md` files in the `www` folder. The pages must only have the content, not the `<html>` and `<body>` tags.

## Markdown Pages

Markdown pages can be written as follows:

```markdown
# This is a title
This is some page content.
```

## Index

The first file in a folder must be called `index.md` or `index.html`. Either file must be present for the system to make this a valid folder to build pages.

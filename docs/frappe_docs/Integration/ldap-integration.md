# LDAP Integration

LDAP integration allows you to authenticate users in Frappe using an LDAP server.

---

## Features

1. **Centralized Authentication**: Manage user authentication through an LDAP server.
2. **Role Mapping**: Map LDAP groups to Frappe roles.
3. **Secure Connection**: Use SSL/TLS for secure communication.

---

## Setting Up LDAP Integration

1. Go to **Integrations** > **LDAP Settings**.
2. Configure the LDAP server details, including the server URL, base DN, and bind credentials.
3. Map LDAP groups to Frappe roles.
4. Test the connection and save the settings.

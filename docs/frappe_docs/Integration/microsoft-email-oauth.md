# Microsoft Email OAuth

Microsoft Email OAuth integration allows you to connect your Microsoft email account to Frappe securely using OAuth.

---

## Features

1. **Secure Authentication**: Use OAuth for secure email authentication.
2. **Email Sending and Receiving**: Send and receive emails directly from Frappe.
3. **Calendar Integration**: Sync your Microsoft calendar with Frappe.

---

## Setting Up Microsoft Email OAuth

1. Go to **Email Account** and select Microsoft OAuth as the authentication method.
2. Configure the client ID, client secret, and redirect URI.
3. Authenticate your Microsoft account to start using the integration.

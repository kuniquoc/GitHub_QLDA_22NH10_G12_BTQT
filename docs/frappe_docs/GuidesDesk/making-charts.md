# Making Charts

Frappe Charts enables you to render simple line, bar, or percentage graphs for single or multiple discrete sets of data points. You can also set special checkpoint values and summary stats.

---

## Learn More

Check out the official documentation for Frappe Charts at [https://frappe.github.io/charts/](https://frappe.github.io/charts/) to learn more about how to create and customize charts.

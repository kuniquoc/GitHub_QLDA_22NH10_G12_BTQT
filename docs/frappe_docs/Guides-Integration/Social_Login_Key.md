# Social Login Key

Add social login providers like Facebook, Frappe, GitHub, Google, Microsoft, etc., and enable social login.

## Setup Social Logins

1. Go to `Integrations > Authentication > Social Login Key`.
2. Select the Social Login Provider or "Custom".
3. If required, enter the "Base URL".
4. Check "Enable Social Login" to show the icon on the login screen.
5. Add the Client ID and Client Secret as per the provider.

### Example

- **Social Login Provider**: Frappe
- **Client ID**: ABCDEFG
- **Client Secret**: 123456
- **Enable Social Login**: Checked
- **Base URL**: `https://erpnext.org` (if required).

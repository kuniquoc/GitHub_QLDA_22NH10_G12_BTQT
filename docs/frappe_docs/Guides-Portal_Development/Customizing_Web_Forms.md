# Customizing Web Forms

Web Forms are a powerful way to add forms to your website. From Version 7.1+, they include tables, paging, and other utilities.

## Standard Web Forms

If you check the "Is Standard" checkbox, a new folder will be created in the module of the Web Form. This folder will contain `.py` and `.js` files that you can use to customize the Web Form.

## Web Form Settings

- **Allow Edit**: Allow each user to have one instance and edit it.
- **Allow Multiple**: Allow users to view and edit multiple instances of the Web Form.
- **Show as Grid**: Show a table view of the Web Form values (only if "Allow Multiple" is set).
- **Allow Incomplete Forms**: Allow users to save without mandatory validation.

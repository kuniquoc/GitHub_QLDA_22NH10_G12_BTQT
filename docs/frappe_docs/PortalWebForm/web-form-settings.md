# Web Form Settings

The **Settings** tab in a Web Form allows you to enable or disable various functionalities.

---

## Key Settings

### 1. **Login Required**
- If disabled, guest users can submit the form.
- If enabled, only logged-in users can submit the form.

### 2. **Allow Multiple Responses**
- Enables users to submit multiple responses.
- Automatically enables the **Show List** checkbox in the List Settings section.

### 3. **Allow Editing After Submit**
- Allows users to edit their submitted responses.

### 4. **Allow Delete**
- Allows users to delete their submitted responses from the Web Form List View.

### 5. **Show List**
- Enables a List View where users can see all their submitted forms.
- Users can only open forms they own unless **Apply Document Permissions** is enabled.

### 6. **Apply Document Permissions**
- Restricts users to only open records they have permission to access.

### 7. **Allow Print**
- Adds a print button to the form in View Mode.

### 8. **Allow Comments**
- Adds a comment section below the form for user communication.

### 9. **Show Attachments**
- Displays all attachments linked to the document in an attachment section.

### 10. **Max Attachment Size (in MB)**
- Restricts users from attaching files larger than the specified size.

### 11. **Allow Incomplete Forms**
- Allows submission of forms even if mandatory fields are not populated.

---

## Sidebar Settings

- **Show Sidebar**: Displays the web form with a sidebar.
- You can select or create a website sidebar to add links to the current web form and other related web forms or pages.

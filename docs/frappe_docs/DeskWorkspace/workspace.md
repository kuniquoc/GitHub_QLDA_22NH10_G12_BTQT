# Workspace

A **Workspace** is the first page you land on when you log in. It also has a dedicated sidebar where you can rearrange the workspace position according to your convenience. The workspace itself is a tool to build user-specific pages. 

---

## Types of Workspaces

### 1. Public Workspaces
- These are listed in the **PUBLIC** section in the sidebar.
- Visible to all users.

### 2. Private Workspaces
- These are listed in the **MY WORKSPACES** section in the sidebar.
- Visible only to the logged-in user (owner).

---

## Creating a Workspace

### Create a New Workspace
1. Click on the **Create Workspace** button.
2. Enter the title and click on **Create**.

### Create a Child Workspace
1. Click on the **Create Workspace** button.
2. Enter the title.
3. Select **Home** in the Parent field and click on **Create**.

---

## Workspace Blocks

Workspace blocks are the building blocks for the workspace page. These blocks can be placed in different variations to build an ideal workspace for a specific use case.

---

## Sidebar

The workspace sidebar has two sections:
1. **MY WORKSPACES**: Contains private workspaces visible only to the logged-in user.
2. **PUBLIC**: Contains standard workspaces visible to all users.

### Notes:
- Only users with the **Workspace Manager Role** can create, edit, or delete public workspaces.

---

## Customization

Workspace has two modes:
1. **Read Only Mode**
2. **Edit Mode**

### Customizing Workspaces
- Start customizing your workspace by clicking on the **Edit** button.
- Use the new workspace builder UI for all kinds of customization.

### Important:
- Adding new blocks from the workspace document is not recommended as the new workspace doesn't know exactly where to add that block.
- The best practice is to use the workspace builder UI for updates and customizations.

# Workspace Blocks

Blocks are the building blocks of the workspace. You can add different types of blocks, edit them, and arrange them as per your needs to build a fully customizable workspace. Currently, there are 9 types of blocks, and more will be added in the future.

---

## Types of Blocks

### 1. **Heading**
- A heading block is used as a header for any section or paragraph.
- It supports an inline toolbar with options like header size (H1 to H6), bold, italic, and links.

---

### 2. **Text**
- A text block is used as a paragraph or description for any section.
- It also supports an inline toolbar with formatting options.

---

### 3. **Card**
- A card contains several links that can be used for quick navigation.
- The link item can be a DocType, Report, or Page.

---

### 4. **Chart**
- You can add any **Dashboard Chart** to the workspace using this block.

---

### 5. **Shortcut**
- A shortcut block is used to quickly navigate to any DocType, Report, or Page.
- Features:
  - Customize the view in which it should open.
  - Add filters that will be applied when opening the selected view.
  - Display the number of records based on the applied filters as a pill.

---

### 6. **Spacer**
- A spacer block is used to add space between two blocks.
- It can also be used to position other blocks by adding white space before or after the block.

---

### 7. **Onboarding**
- The onboarding block is used to guide first-time users when they land on your page.
- Features:
  - Steps to follow to learn a specific module.
  - Descriptions or links to videos for each step.

---

### 8. **Quick List**
- The quick list block displays recently updated records of a selected DocType.
- Features:
  - Add filters to refine the list.
  - In read-only mode, you can:
    - Refresh the list.
    - Update applied filters.
    - Create a new record.
    - Open the DocType's list view.

---

### 9. **Number Card**
- You can add any existing **Number Card** to the workspace using this block.

---

## Customization

- Blocks can be added, edited, and rearranged to create a personalized workspace.
- Use the workspace builder UI to manage blocks effectively.

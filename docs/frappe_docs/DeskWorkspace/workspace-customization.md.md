# Workspace Customization

Workspaces in Frappe can be customized to suit your needs. Customization can be done in two main areas: **Customizing Workspaces** and **Customizing Workspace Pages**.

---

## Customizing Workspaces

All workspace customizations can be done through the sidebar, except for creating a new workspace. Below are the available customization options:

### 1. Edit Workspace Details
- Update the title and icon.
- Set the workspace as a parent or child.
- Make the workspace public or private.

### 2. Create Duplicate
- Duplicate an existing workspace.
- Add or remove blocks as needed in the duplicate.

### 3. Delete Workspace
- Delete an existing workspace.

### 4. Hide Workspace
- Hide private workspaces.
- Only users with the **Workspace Manager** role can hide public workspaces.

### 5. Rearrange Position
- Drag and drop workspaces in the sidebar to change their position.
- You can also make a workspace a child or parent by rearranging it.

---

## Customizing Workspace Pages

Customizing blocks within a workspace falls under this category. Below are the available options:

### 1. Add a New Block
- Add new blocks to the workspace page.

### 2. Edit an Existing Block
- Modify the content or settings of an existing block.

### 3. Remove or Delete a Block
- Remove or delete blocks from the workspace page.

### 4. Resize the Width of a Block
- Adjust the width of blocks to fit your layout.

### 5. Rearrange the Position of a Block
- Drag and drop blocks to rearrange their position on the workspace page.

---

## Workspace Modes

Workspaces have two modes:
1. **Read Only Mode**
2. **Edit Mode**

To start customizing a workspace, click on the **Edit** button. Note that customizing workspaces and customizing workspace pages are two different things.

---

## Save and Discard Changes

- Use the **Save** or **Discard** buttons to save or discard workspace page customizations.
- Changes made to the workspace via the sidebar are saved automatically.

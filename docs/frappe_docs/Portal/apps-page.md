# Apps Page

The Apps Page in Frappe allows you to showcase and manage applications available in your system. It is useful for providing users with a centralized location to access and explore apps.

---

## Features

1. **App Listings**: Display all available apps with descriptions and icons.
2. **Search and Filters**: Allow users to search for apps or filter them by category.
3. **Customizable Layout**: Use templates to design the apps page.

---

## Setting Up the Apps Page

1. Go to the **Apps Page** list.
2. Click on **New** to create a new apps page.
3. Set the following:
   - **Title**: The title of the page.
   - **Route**: The URL path for the page.
   - **Apps**: Add the apps you want to display on the page.
4. Save the page.

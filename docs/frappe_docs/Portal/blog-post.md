# Blog Post

The Blog Post feature in Frappe allows you to create and manage blog articles for your website. It is a simple and effective way to share updates, tutorials, or announcements.

---

## Creating a Blog Post

1. Go to the **Blog Post** list.
2. Click on **New** to create a new blog post.
3. Fill in the following fields:
   - **Title**: The title of the blog post.
   - **Blog Category**: Select the category for the blog post.
   - **Content**: Write the content of the blog post using the rich text editor.
   - **Published On**: Set the publication date.
4. Save the blog post.

---

## Features

- **SEO Optimization**: Add meta tags and descriptions for better search engine visibility.
- **Categories**: Organize your blog posts into categories.
- **Draft Mode**: Save posts as drafts before publishing.

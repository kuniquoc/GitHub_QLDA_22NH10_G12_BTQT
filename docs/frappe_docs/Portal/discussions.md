# Discussions

You can now add a discussions component on any web page of the portal. The discussions component allows logged-in users to enter and share messages with other users. This component can be used for discussions or to set up a portal where users help each other with their queries.

---

## Steps to Setup the Discussions Component

1. Open the web page document on which you want to add the discussions component or create a new web page and save it.
2. Scroll to the **Page Building Blocks** table.
3. Under **Web Template**, select **Discussions**.
4. Click on **Edit Values**.
5. Enter a **Title**, **CTA Title**, and select the Web Page as the current web page name.
6. Check **Single Thread** if you want the discussion to be single-threaded.
7. Save the page.

# Portal Pages

Portal Pages in Frappe allow you to create and manage web pages that are accessible to users through the portal. These pages can display dynamic content and provide user-specific information.

---

## Features

1. **Customizable Layouts**: Use templates and building blocks to design your portal pages.
2. **Dynamic Content**: Display user-specific data like orders, invoices, or tasks.
3. **Authentication**: Restrict access to certain pages based on user roles.

---

## Creating a Portal Page

1. Go to the **Portal Page** list.
2. Click on **New** to create a new portal page.
3. Set the following:
   - **Title**: The title of the page.
   - **Route**: The URL path for the page.
   - **Role**: Restrict access to users with specific roles.
4. Add content using the **HTML Editor** or **Building Blocks**.

# Bench Commands

Bench provides a variety of commands to manage your Frappe environment.

---

## Common Commands

1. **Create a New Site**:
   ```bash
   bench new-site [site-name]
   ```

2. **Install an App**:
   ```bash
   bench --site [site-name] install-app [app-name]
   ```

3. **Start Bench**:
   ```bash
   bench start
   ```

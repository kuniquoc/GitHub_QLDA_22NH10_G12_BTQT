# Frappe Commands

Frappe commands allow you to perform specific tasks related to Frappe apps and sites.

---

## Common Commands

1. **Run Tests**:
   ```bash
   bench run-tests
   ```

2. **Backup Site**:
   ```bash
   bench backup
   ```

3. **Migrate Site**:
   ```bash
   bench migrate
   ```

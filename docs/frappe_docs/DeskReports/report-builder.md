# Report Builder

The Report Builder in Frappe allows you to create simple reports without writing any code. It is ideal for users who need quick access to data without requiring advanced customization.

---

## Features

1. **Drag-and-Drop Interface**: Add fields to your report by dragging them from the sidebar.
2. **Filters**: Apply filters to refine the data displayed in the report.
3. **Export Options**: Export the report to Excel or CSV formats.
4. **Charts**: Add visualizations to your report.

---

## Creating a Report

1. Navigate to the **Report Builder**.
2. Select the DocType you want to report on.
3. Add fields, filters, and groupings as needed.
4. Save the report for future use.

---

## Limitations

- The Report Builder is limited to a single DocType.
- For more complex reports, use **Query Reports** or **Script Reports**.

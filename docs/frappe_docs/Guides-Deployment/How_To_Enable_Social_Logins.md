# How To Enable Social Logins

Enable social login providers like Facebook, Google, or GitHub to allow users to log in without remembering another password.

## Steps to Enable Social Logins

1. Obtain the Client ID and Client Secret from the authentication service.
2. Go to `Desktop > Integration > Authentication > Social Login Key`.
3. Create a new Social Login Key and fill in the following details:
   - **Social Login Provider**: Select the provider (e.g., Google, Facebook).
   - **Client ID**: Enter the Client ID.
   - **Client Secret**: Enter the Client Secret.
   - **Base URL**: Enter the base URL if required.
4. Check "Enable Social Login" to activate the feature.
